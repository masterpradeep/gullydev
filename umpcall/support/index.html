Privacy Terms

What UmpCall does

Works offlineDigital toss, umpire counter, and solo scorebook do not require login.
Club cloudShared rosters, roles, match handover, archives, and player stats.
Live followSpectators enter a match code to watch a live scoreboard.

What to include

  • Device model, OS version, and UmpCall version from Settings, About.
  • Steps to reproduce the issue.
  • Whether you were using club/live features or scoring offline.
  • Screenshots or screen recording, if useful.

Common questions

Create a clubSettings, Club, Connect your club, Create. Save the Club ID and scorer join code.
Recover adminUse the saved Club ID and scorer join code. Viewer codes cannot recover admin access.
Follow matchThe scorer publishes a live match code. Spectators enter that code from Watch Live Match.
No email loginCloud features use an anonymous device session, not an email/password account.

Data and deletion help

Local dataDelete matches in the app or uninstall UmpCall to remove device data.
Club dataLeave a club or disband it as admin. For server erasure, email with subject Privacy.
Sync helpIf club changes do not appear, check your connection and reopen the club while online.
Offline testAirplane mode can test toss, counter, and solo scorebook.
